Scoreboard App

About the App

Scoreboard is a simple and functional app for keeping track of player scores during games. Add players, update scores, and track progress effortlessly.

Features

  • Add players with custom names and avatars.
  • Track scores with easy increment and decrement buttons.
  • View game history and set game settings like starting score and points per tap.
  • Dark mode support.
  • Option to allow negative scores.
